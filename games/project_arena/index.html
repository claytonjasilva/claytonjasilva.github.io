🎯 Project Arena

Plataforma de Simulação para Gestão de Projetos

Bem-vindo

O Project Arena é um ambiente gamificado para apoio ao ensino de Gestão de Projetos. Nesta primeira versão, uma equipe participa de desafios conduzidos por um árbitro.

👨‍🏫 Entrar como Árbitro 👥 Entrar como Equipe

Objetivos

  • Tomada de decisão.
  • Raciocínio crítico.
  • Aplicação dos conceitos.
  • Aprendizagem baseada em desafios.
  • Evolução por fases.

Versão

Versão 1.0 (Protótipo)

Projeto desenvolvido para apoio às disciplinas de Engenharia.
