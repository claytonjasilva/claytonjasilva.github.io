Passo a passo no MS Project

Como Executar no MS Project

Criar novo projeto
Arquivo → Novo → Projeto em branco. Configure as informações do projeto: Projeto → Informações do Projeto. Defina a data de início (ex: 02/Jan/2025).
Inserir tarefas resumo e subtarefas
Digite os nomes das tarefas na coluna "Nome da Tarefa". Para criar a hierarquia EAP, selecione as subtarefas e clique em Tarefa → Recuar Tarefa (ou use Alt+Shift+→). Tarefas resumo aparecem em negrito automaticamente.
Inserir durações
Clique na coluna Duração e insira o valor (ex: "4 dias" ou "4d"). O MS Project calcula datas automaticamente com base no calendário do projeto.
Definir dependências
Na coluna "Predecessoras", insira o número da tarefa predecessora. Tipo padrão: Término-Início (FI/FS). Para outros tipos: duplo-clique na seta do Gantt → Dependência de Tarefa.
Visualizar o Gráfico de Gantt
O Gantt é a exibição padrão. Use Exibição → Gráfico de Gantt para confirmar. Barras azuis = tarefas normais; barras pretas = resumo; losango preto = marco.
Verificar o caminho crítico
Exibição → Gráfico de Gantt → Ferramentas de Gráfico de Gantt → Formatar → marque "Caminho Crítico". Tarefas críticas ficam vermelhas.
Salvar e exportar
Arquivo → Salvar como (.mpp). Para exportar o Gantt como imagem: Arquivo → Imprimir → configurar para imprimir apenas o Gantt. Ou Copiar Imagem (Tarefa → Copiar → Copiar Imagem).
⚠️ Atenção
No MS Project, tarefas em modo automático (Agendado automaticamente) respeitam as dependências. Tarefas em modo manual permitem datas livres. Para este exercício, use sempre o modo automático (Tarefa → Modo de Agendamento → Agendado Automaticamente).
