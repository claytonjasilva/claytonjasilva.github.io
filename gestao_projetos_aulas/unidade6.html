6.1 / processos

Processos de Gestão dos Custos

7.1
Planejar o Gerenciamento dos Custos
Define como os custos serão estimados, orçados, gerenciados, monitorados e controlados.
7.2
Estimar os Custos
Desenvolve uma estimativa dos recursos monetários necessários. Técnicas: análoga, paramétrica, bottom-up.
7.3
Determinar o Orçamento
Agrega os custos individuais das atividades para criar a linha de base dos custos. Inclui reservas de contingência.
7.4
Controlar os Custos ⚡
Monitora o status do projeto para atualizar os custos e gerenciar desvios. Usa o EVM (Earned Value Management).

Tipos de Custos

Diretos
Atribuídos diretamente ao projeto: mão-de-obra, materiais específicos, equipamentos.
Indiretos
Compartilhados entre projetos: overhead, aluguel, energia, administração.
Fixos
Não variam com o volume de trabalho: licenças de software, aluguéis.
Variáveis
Variam com a quantidade de trabalho executado: horas/homem, materiais consumíveis.
Contingência
Reservas para riscos identificados ("known unknowns").
Gerencial
Reservas para riscos desconhecidos ("unknown unknowns"). Acima da linha de base.

6.2 / earned value management (evm)

Gerenciamento do Valor Agregado — EVM

Definição
O EVM é uma metodologia que integra escopo, cronograma e custo para avaliar o desempenho e progresso do projeto de forma objetiva, comparando o planejado com o realizado.

Variáveis Fundamentais do EVM

PV — Planned Value
Valor Planejado
Orçamento autorizado para o trabalho agendado até a data de referência.
EV — Earned Value
Valor Agregado
Valor do trabalho realmente concluído até a data de referência, medido pelo orçamento aprovado.
AC — Actual Cost
Custo Real
Custo total incorrido para o trabalho executado até a data de referência.
BAC — Budget at Completion
Orçamento na Conclusão
Orçamento total aprovado para o projeto ou atividade.

Variações e Índices

CV — Cost Variance
Variação de Custo
CV = EV − AC
CV > 0 → abaixo do orçamento ✅
CV < 0 → acima do orçamento ❌
SV — Schedule Variance
Variação de Prazo
SV = EV − PV
SV > 0 → adiantado ✅
SV < 0 → atrasado ❌
CPI — Cost Performance Index
Índice de Desempenho de Custo
CPI = EV / AC
CPI > 1 → eficiente em custo ✅
CPI < 1 → ineficiente em custo ❌
SPI — Schedule Performance Index
Índice de Desempenho de Prazo
SPI = EV / PV
SPI > 1 → adiantado ✅
SPI < 1 → atrasado ❌
