Prática 4
Entrada binária de 3 bits

Prof. Clayton J A Silva Arduino Mega 2560 Rev 2026.2

Objetivo

  • Ler três chaves como uma palavra binária de 3 bits e converter o valor de entrada para decimal no Monitor Serial.
Materiais Montagem Sketchs propostos Novo shield

I / instrumental e ferramental

Materiais utilizados

  • Protoboard
  • LED — Diodo Emissor de Luz
  • Resistor de 300 Ω
  • Arduino Mega 2560
  • Switch / chave
Atenção: mantenha os pinos de menor numeração associados aos bits menos significativos, para facilitar a leitura da palavra binária no circuito.

II / montagem

Entrada binária de três bits

O circuito utiliza três chaves como entradas binárias. As saídas das chaves devem ser conectadas aos pinos digitais 2, 3 e 4 do Arduino.

Figura 1 — Representação em SVG do circuito com três chaves conectadas ao Arduino como palavra binária de 3 bits.

  1. Monte o circuito com três chaves ligadas às entradas digitais do Arduino.
  2. Escreva um código para converter a entrada binária de 3 bits em decimal.
  3. Observe o resultado pelo Monitor Serial da IDE do Arduino.

III / novos sketchs

Elaborar novos sketchs para o mesmo shield

1. Soma dos pares não negativos inferiores ao número lido

O sistema deve ler a palavra código de entrada. Caso ela corresponda a um número par, o sistema deverá ler via Monitor Serial um número inteiro, calcular a soma dos números pares não negativos inferiores ao número lido e escrever o resultado no Monitor Serial.

Sugestão de solução — III.1

2. Contagem binária crescente cíclica

O sistema deve apresentar no Monitor Serial uma contagem binária crescente e cíclica a partir do código lido até 111.

Exemplo: entrada = 011 saída: 011 → 100 → 101 → 110 → 111 → 011 → 100 → ...
Sugestão de solução — III.2

IV / novo shield

Elaborar um novo shield

O projeto deverá ativar saídas específicas conforme o código binário de entrada.

Figura 2 — Proposta em SVG: três chaves de entrada e saídas para LEDs, buzzer e Monitor Serial.

EntradaAção
000Ativa um LED vermelho
001Ativa um LED amarelo
010Ativa um buzzer com frequência de 100 Hz durante 2 s, intermitente, em intervalos de 5 s
011Ativa um buzzer com frequência de 50 Hz durante 3 s, intermitente, em intervalos de 2 s, e ativa um LED verde
Demais casosEscreve no Monitor Serial o número decimal equivalente
