Trabalho AP1
Entrada e saída binária de 6 bits

Prof. Clayton J A Silva Arduino Mega 2560 Rev 2026.2

Objetivo

  • Projetar um sistema embarcado com Arduino Mega 2560 capaz de processar uma palavra binária de 6 bits e produzir uma saída binária de 6 bits conforme operações aritméticas definidas.
Condições Problema Operações Testes Entregáveis

I / condições gerais

Condições gerais

  1. O trabalho corresponde à parte 2 da AP1.
  2. O trabalho deve ser realizado e submetido em grupo.
  3. A entrega do pedido será realizada presencialmente.
O arquivo original indicava data de entrega em 10 de outubro de 2022. Recomenda-se atualizar essa data antes da publicação para a turma atual.

II / dados do problema

Projeto do sistema embarcado

Projetar um sistema embarcado baseado na placa Arduino Mega 2560, com uma palavra binária de entrada de 6 bits e uma palavra binária de saída de 6 bits.

Figura 1 — Representação em SVG do sistema: entrada E[5:0], processamento no Arduino e saída S[5:0] em LEDs.

  • Entrada: E(E5 E4 E3 E2 E1 E0).
  • Saída: S(S5 S4 S3 S2 S1 S0).
  • Utilizar pinos de entrada adjacentes, com pinos de menor numeração associados aos bits menos significativos.
  • Cada pino de saída deve ser conectado a um LED.

III / operações do sistema

Operações básicas

EntradaMnemônicoOperação
0x00 a 0x0FaddAdição imediata
0x10 a 0x1FsubSubtração imediata
0x20 a 0x2Fadd xAdição direta
0x30 a 0x3Fsub ySubtração direta

Descrição das operações

  1. add: soma os bits E3 E2 + E1 E0 e apresenta o resultado na saída.
  2. sub: subtrai E3 E2 − E1 E0 e apresenta o resultado em complemento de 2.
  3. add x: soma E3 E2 E1 E0 a uma variável não negativa x, definida no código.
  4. sub y: subtrai de E3 E2 E1 E0 uma variável não negativa y, definida no código.
Em caso de resultado superior à capacidade de representação da saída, o sistema deve indicar overflow piscando todos os LEDs.

IV / testes de verificação

Setup de teste

x = 34d y = 48d modo = 2

Implementar e testar a sequência de instruções em notação simbólica:

add, 0x7 sub y, 0xF add x, 0xB sub, 0xC

A variável modo deve indicar a representação da saída na operação de subtração direta:

modoRepresentação
0Bit sinal
1Complemento de 1
2Complemento de 2
3Excesso

V / pedido

Entregáveis

  1. Memorial descritivo dos componentes utilizados.
  2. Esquema elétrico geral do circuito, indicando todas as ligações realizadas.
  3. Protótipo do sistema com base na placa Arduino Mega 2560.
  4. Código elaborado para a IDE do Arduino.
  5. Relatório descrevendo o resultado do teste proposto e analisando se o resultado é o esperado.
  6. Elaboração de pelo menos mais dois testes em notação simbólica, com descrição dos resultados observados.
