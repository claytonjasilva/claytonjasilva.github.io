2. Requer polarização

| Característica | Valor |
| - | - |

| Som de saída | ≥ 85 dB | 
| Frequência de Resonância | 2300 ± 300Hz | 
| Temperatura de Operação | -20ºC ~ +45ºC | 
| Temperatura de armazenamento | -20ºC ~ +60ºC |
